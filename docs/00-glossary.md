# Glossary

| Term | Meaning | Note |
|------|---------|------|
| User | 利用者 | このアプリを使う人 |
| Page | ページ | メモの単位。タイトルと複数のブロックを持つ |
| Block | ブロック | ページを構成する最小単位（段落、見出し、リストなど） |
| Workspace | ワークスペース | ユーザーが作成したページの集合 |
| Draft | 下書き状態 | 保存されているが公開されていない状態 |
| Published | 公開状態 | 他のユーザーと共有可能な状態（MVP外） |
| TBD | To Be Determined | 未決定事項 |
